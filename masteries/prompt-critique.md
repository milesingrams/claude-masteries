## Chip

Claude could help you ask this better

## Triggers

- User's prompt is vague or underspecified
- User seems to be struggling to articulate what they want
- Request is complex and might benefit from restructuring
- User is asking something for the first time
- Prompt could be clearer or more effective

## Satisfaction

- User asks Claude to improve or critique their prompt
- User asks how to ask something more effectively
- Phrases: "how should I ask", "improve this prompt", "what's a better way to ask", "help me phrase", "critique my question", "how could I make this clearer"

## Detail

Claude can help you prompt Claude. If you're not sure how to ask something, ask Claude to improve your question before answering it. You'll learn what makes prompts effective while getting better results. Meta-prompting is a skill that compounds.
