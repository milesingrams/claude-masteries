## Chip

You could ask Claude what's possible here

## Triggers

- User seems unsure how to approach a problem
- User is asking something that could be done multiple ways
- User might not know what Claude can do
- Task is open-ended or exploratory
- User is new to a type of request

## Satisfaction

- User asks Claude what it can do or how it would approach something
- User asks Claude what it needs to help effectively
- Phrases: "what can you do", "how would you approach", "what do you need from me", "what are my options", "how could you help with", "what's possible"

## Detail

When you're not sure what's possible, ask. Claude can explain its capabilities, suggest approaches, or tell you what information would help it help you. Turning Claude into a guide to its own abilities unlocks possibilities you didn't know existed.
