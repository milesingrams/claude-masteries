## Chip

You could show Claude the file directly

## Triggers

- User is describing or summarizing a document
- User is asking about content they have in a file
- User is transcribing or copy-pasting content
- User references a document, PDF, or data file
- Task involves analyzing, summarizing, or working with existing content

## Satisfaction

- User uploads or attaches a file to the conversation
- User shares a document directly rather than describing it
- Phrases: "here's the file", "I've attached", "see attached", "uploading", "let me share the document"
